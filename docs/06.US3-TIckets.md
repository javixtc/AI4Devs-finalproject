## Phase 3  Application

### GEN-T005  Application: GenerateMeditationUseCase
**Type**: Task · **Status**: Completed
Synchronous orchestration:
1. Fetch Meditation from meditationbuilder via FindMeditationPort
2. Generate voice (TTS) via GenerateVoicePort
3. Generate background image via GenerateImagePort
4. Render video (FFmpeg) via RenderVideoPort
5. Store assets in S3 via StoreMediaPort
6. Persist GenerationJob with final URLs via PersistGenerationPort
Returns GenerationResult(videoUrl, audioUrl, subtitlesUrl).
