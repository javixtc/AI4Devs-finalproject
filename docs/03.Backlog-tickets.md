## 🎯 Why the final effort exceeded initial estimates

- **FFmpeg Complexity**: Synchronizing audio, video, and subtitles while ensuring valid S3 uploads took longer than anticipated.
- **Security Migration**: Retrofitting existing BCs to use the new Identity context required more refactoring than expected.
- **AI Prompt Tuning**: Achieving consistent results from OpenAI for both text and background images required multiple iterations of domain services.
- **Testing Lifecycle**: The 8 backend CI gates and 6 frontend gates provided high reliability but increased the development cycle per ticket.
